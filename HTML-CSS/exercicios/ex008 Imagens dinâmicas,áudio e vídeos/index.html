Preload

Indica se o áudio será ou não pré-carregado e aceita 3 valores:

  • metadata - carrega apenas as informações sobre o arquivo (tamanho,tempo,informações de direitos etc)
  • none - não vai carregar nada até dar play.
  • auto - é o padrão e vai carregar o arquivo inteiro assim que o usuário entrar no site, mesmo que nunca dê o play.
